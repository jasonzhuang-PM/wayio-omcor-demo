# 官網雛形｜執行與部署說明

> 對應週一量化指標:官方網站雛形 1 個可執行版本,客戶名稱已匿名化(以案例 A/B/C 呈現)。

## 內容
- `index.html` — 單檔靜態網站,含首頁、解決方案、案例(匿名)、合作流程、關於我們、聯絡表單。無需建置工具,無外部相依。

## 本地執行(最快)
直接用瀏覽器打開 `index.html` 即可預覽。

## 部署到 Vercel(規格書指定帳號)
1. 把 `05_官網雛形/` 資料夾推上 GitHub(或整個 MCUT PM repo)。
2. Vercel → New Project → 匯入該 repo。
3. Framework Preset 選 **Other**;Root Directory 指向 `05_官網雛形`;Output 為靜態,無需 build 指令。
4. Deploy,取得預覽網址,填回 `週一記事本` 與 `官方網站_了解與雛形說明.md`。
